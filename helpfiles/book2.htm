Introduccion

El generador de datos es una sencilla herramienta para insertar datos en mysql de forma que no tengamos que estar insertando datos de manera manual, lo que nos llevaria bastante tiempo como para que despues eliminemos esos datos de prueba y tengamos que empezar de nuevo. Funciona de manera simple, de momento solo esta pensada para este gestor de base de datos que es MySQL. EL programa esta hecho en python y fue pensado para desarrollo en linux. aunque tambien puede ser usada en MAC o Windows si se tiene instalado python y wxwidgets.

Para user el generador solo es necesario hacer unos cuantos clicks y si quermos despues lo podemos hacer manualmente.

  • Hay que especificar las variables de conexion a la base de datos de mysql
  • Enseguida seleccionar la base de datos a la que queremos llenar
  • Elegir los datos aleatorios tipo hasta imitar los campos a llenar en la bd
  • Generar datos


Menu

  • Archivo:
    • QUIT: Salir de la aplicacion
  • Extensiones:
    • Nueva: Nueva lista de la que se obtendran datos aleatorios
    • Listar: Lista de datos aleatorios
  • Database:
    • Conectar: Lanza una ventana con los datos para la conexion a mysql, como host, user y password
    • Databases: lista las bases de datos existentes y permite elegir la base de datos
  • Help:
    • Ayuda: Muestra el archivo de ayuda
    • A cerca de: Muestra la informacion del programa
